# Trello Power-Up Examples

This directory contains example files from the original Glitch Trello Power-Up template. These demonstrate various Power-Up capabilities but are not used in the Book Detail Power-Up.

## Example Files

### JavaScript Examples
- **orig-client.js** - Original complete template client
- **section.js** - Card back section example (Yellowstone attachment)
- **modal.js** - Modal resize demonstration
- **board-bar.js** - Board bar navigation example
- **settings.js** - Power-Up settings page example
- **celebrate.js** - Confetti animation demo

### HTML Pages
- **section.html** - Card back section page
- **modal.html** - Modal demonstration page
- **board-bar.html** - Board bar page
- **settings.html** - Settings page
- **celebrate.html** - Celebration/confetti page
- **authorize.html** - Authorization flow example
- **api-client-authorize.html** - API client authorization
- **auth-success.html** - Auth callback page

### CSS
- **card-back-section.css** - Debug styles with dashed outlines
- **board-bar.css** - Board bar styling
- **modal.css** - Modal styling
